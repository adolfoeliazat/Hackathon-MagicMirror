## Installation
* Install nodejs

```
$ brew install nodejs
```

* Install all dependencies (both npm and bower packages):

```
$ npm install
```

* Run the local development server

```
$ npm run serve
```


## Development mode
```
$ npm run serve
```

## Production mode
```
$ npm run gulp production
$ node.server.js
```
This builds the app for production and serves it and the request/recommend/scan API
via node.
